BlueShoes NumberField Example 2

This example demonstrates the Number Field component.

  • Using the "drawButtonsInside" feature. The buttons look like they're inside the number fields.
  • Using the "zeroFill" feature. If the hour is below 10, it will be shown as for example 09, not 9.
  • You might have notices it: The fields show your systems time of the moment when the page loaded.
  • Using the TitleToTooltip feature. Click into a field to see it.
