JavaScript Bs_Dropdown example 3

  • Similar to example 1.
  • Uses an attached onChange event that alerts the new value.
  • Since myDropdown.fieldName is not defined, the field name of the hidden field is "myDropdown_value". This is the value you should use from the submitted form data. "myDropdown_input" and "myDropdown_select" are only the helper fields, they may not be up to date when submitting the form.
  • The width is defined using setSize().
