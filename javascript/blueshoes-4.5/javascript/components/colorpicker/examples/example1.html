BlueShoes ColorPicker Example

This is an example of the BlueShoes Color Picker.

Color 1:
  • Standard color picker.
 
Color 2:
  • Using the colorizeBackground feature here.
  • A default value is defined using setColorByHex().
 
Color 3:
  • Different button icon and title are used.
  • A default value is defined in the html tag.
  • Attached an 'onAfterChange' event that alerts the value.
